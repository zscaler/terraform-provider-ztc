# Changelog

## 0.1.0 (October 3, 2025)

### Notes

- Release date: **(October 3, 2025)**
- Supported Terraform version: **v1.x**

This new Provider (ZTW Terraform Provider) enables fully automated provisioning and configuration across several key components in the Cloud & Branch Connector portal.
